## Specialties

**API and developer documentation**
I've documented REST APIs, GraphQL endpoints, WebSocket integrations, and SDKs. I work directly from OpenAPI/Swagger specs and can write or review spec files alongside docs. 

**User guides and manuals**
From onboarding flows to feature-level how-to articles, I write user documentation that reduces support burden and improves product adoption.

**SaaS product documentation**
I've supported SaaS documentation at scale — managing content in docs-as-code environments, writing release notes and changelogs, and ,maintaining knowledge base content.

**AI documentation tooling**
I build reusable prompt libraries and AI workflows that accelerate documentation production. My GitHub repository includes a working prompt library for VS Code + GitHub Copilot that automates common doc tasks including FAQ generation, user guide drafts, and API doc scaffolding.
